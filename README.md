## 下载

### Docker

```
docker pull apachecn0/git-doc-zh
docker run -tid -p <port>:80 apachecn0/git-doc-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g git-doc-zh
git-doc-zh <port>
# 访问 http://localhost:{port} 查看文档
```
